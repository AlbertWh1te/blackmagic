意图识别输出说明文档

输出类型:
    1. list
        输出的数组为分词加词性之后的结果,表示无意图,进入匹配
    2. dict
        输出的字典为意图具体信息,表示获取意图,进入意图抓取
        字典中具体信息
            键名            数据类型            数据备注
            'quest':        数组,               疑问代词列表(非必要)
            'ask':          数组,               询问代词列表(非必要)
            'loc':          字典,               地点信息字典
                                                若是天气意图: {start: 天气城市}
                                                若是火车意图: {start: 出发地, end: 目的地}
            'time':         字符串,           具体时间所代表的年月日,并以"-"隔开
            'intention':    字符串,             意图类型, 天气为weather, 火车为train
            'details':      字典                (非必要)
            'length':       整数,               句子长度(非必要)
            'timediff':     datetime,           具体时间与今天的时间间隔
    3. None
        输出为None, 表示需要提醒用户重新输入


涵盖时间功能:
    1. 不同形式时间的输入
    2. 部分公历节假日时间
    3. 部分农历节假日时间
